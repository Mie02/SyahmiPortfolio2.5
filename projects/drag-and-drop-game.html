Drag & Drop Learning Game

A Unity-based interactive learning game focused on drag-and-drop matching with feedback. Designed for clarity, smooth UI flow, and replayability.

UnityC#UI/UXGame LogicFeedback

Step 1 — Drag & Drop

Players drag answers into slots. System checks if all slots are filled.

Step 2 — Validation

Submit button validates each slot and gives correct/wrong feedback.

Step 3 — Scoring & Timer

Points and timer create challenge; difficulty can change background/UI.

Highlights

Clean UI Flow
Easy to understand layout and button actions.
User Feedback
Immediate messages when slots are missing or incorrect.
Expandable
Can add hints, difficulty themes, and new question sets.
